# pmenu

domain aware passmenu wrapper script using [TabFS](https://github.com/osnr/TabFS)

My password store is structured like so, and this code assumes that to be the case.

Password Store  
├── example.com  
│   └── user@example.com  
├── example.ca  
│   └── user@example.ca  
├── example.org  
│   └── user@example.org  
└── example.net  
    └── UserName
